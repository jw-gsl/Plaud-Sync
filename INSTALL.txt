==============================================================
 Plaud Sync — Install & Use
==============================================================

Plaud Sync downloads your Plaud recordings (and their transcripts)
to a local folder on your computer, so you can work on them
directly — including in Claude — without emailing or re-uploading.


--------------------------------------------------------------
INSTALL (Windows)
--------------------------------------------------------------
1. Double-click the installer (the ".._x64-setup.exe" or ".msi"
   in this folder).

2. If Windows shows "Windows protected your PC" (SmartScreen),
   this is because the build is not code-signed. Click:
        More info  ->  Run anyway
   (Safe to proceed for a build you trust.)

3. Follow the prompts. If the Microsoft Edge "WebView2" runtime
   isn't already installed, the installer will fetch it.

4. Launch "Plaud Sync" from the Start menu.

INSTALL (macOS)
   Open the .dmg and drag "Plaud Sync.app" to Applications.
   If macOS blocks the first launch, right-click the app -> Open,
   or run:  xattr -cr "/Applications/Plaud Sync.app"


--------------------------------------------------------------
FIRST RUN
--------------------------------------------------------------
1. Sign in: click "Continue with Plaud" and sign in through the
   Plaud website (Google, Apple, email, etc.). If sign-in misbehaves,
   use the "Token" option (paste a token from web.plaud.ai DevTools)
   or email/password.

2. Choose a save folder when prompted (or in Settings).

3. Click "Sync" to refresh the recordings list, then "Download" to
   save the audio (+ transcript .txt). Click a downloaded row to
   reveal the file in File Explorer / Finder.


--------------------------------------------------------------
RUN IT AS A BACKGROUND SYNC TOOL (optional)
--------------------------------------------------------------
In Settings:
  - Auto-sync new recordings (+ interval, e.g. 15 min)
  - Start at login
  - Start minimized


--------------------------------------------------------------
WORK ON TRANSCRIPTS IN CLAUDE (the point of syncing)
--------------------------------------------------------------
Plaud Sync saves each transcript as a .txt next to the audio, in a
local folder Claude can read directly — no Plaud -> email -> upload.

Set it up once:
  1. In Settings, set the save folder to a dedicated location, e.g.
        Windows:  C:\Users\<you>\PlaudTranscripts
        macOS:    ~/PlaudTranscripts
  2. Enable "Download transcript" and "Auto-sync" so new transcripts
     land there automatically.

Then work on them in Claude:
  - Claude Code: open a terminal in the sync folder and run:
        claude
    It reads the .txt transcripts in place — ask it to summarise a
    meeting, extract action items, draft a follow-up, compare calls,
    etc. No upload step.
  - Or point your editor's Claude integration at that folder so it's
    always available.

Net effect: record -> it auto-syncs to the folder -> the transcript
is already sitting where Claude can use it.


--------------------------------------------------------------
UNINSTALL
--------------------------------------------------------------
Windows: Settings -> Apps -> Installed apps -> Plaud Sync -> Uninstall.
Your downloaded recordings are NOT removed.


--------------------------------------------------------------
NOTES
--------------------------------------------------------------
- Unsigned build: SmartScreen/Gatekeeper/antivirus may warn.
- WebView2 (Windows) is required; the installer bootstraps it.
- Your tokens are stored in the OS keychain / Credential Manager;
  recordings go to the folder you choose.
